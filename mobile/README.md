# CurrentDao Mobile

React Native implementation for full energy trading workflow:
- offline trading
- native camera + GPS + biometric
- push notifications
- iOS 14+ and Android 8+ support
- performance goal: <2 seconds cold start

## Setup

1. `cd mobile`
2. `npm install`
3. `npm run ios` / `npm run android`

## Structure

- `src/components`
- `src/screens`
- `src/services`
- `src/native`

## Testing

- `npm test`

## Build steps

Refer to iOS and Android platform docs in `mobile/ios` and `mobile/android`.
